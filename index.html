हिन्दी Quotes

GitHub: https://github.com/nkilm/hindi-quotes

Usage

GET /random

URL: https://hindi-quotes.vercel.app/random

Returns a random quote(Object) of any type.

                  
                    {
                        "type": "positive",
                        "quote": "जिंदगी में कुछ नेक काम ऐसे भी करने चाहिए जिनका, उस ऊपर वालें के सिवा कोई दुसरा गवाह ना हो।"
                    }

GET /random/{type}

URL: https://hindi-quotes.vercel.app/random/success

Returns a quote(Object) of given type. E.g. GET /random/success.

                  
                    {
                        "type": "success",
                        "quote": "जबतक आप खुद पर विश्वास नहीं करते, तबतक आप उपरवालें पर भी विश्वास नहीं कर सकते…"
                    }
