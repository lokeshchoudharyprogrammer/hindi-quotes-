हिन्दी Quotes REST API

Usage

GET /random

Response from API

          
            {
                "type": "positive",
                "quote": "जिंदगी में कुछ नेक काम ऐसे भी करने चाहिए जिनका, उस ऊपर वालें के सिवा कोई दुसरा गवाह ना हो।"
            }

GET /hindiquotes/{type}

Returns a quote(Object) of type {type} quotes e.g. GET /api/quotes/5.

          
            {
                "type": "success",
                "quote": "जबतक आप खुद पर विश्वास नहीं करते, तबतक आप उपरवालें पर भी विश्वास नहीं कर सकते…"
            }
